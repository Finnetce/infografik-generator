Infografik-Generator
Artikel einfügen, Format wählen, Grafik generieren. Branding und Layout werden automatisch angewendet.
Setup: Die Webhook-URL wird in der Variable WEBHOOK_URL im Script konfiguriert. Aktuell: Make.com Szenario-Webhook. Das Frontend kann lokal geöffnet, auf einem Server gehostet oder als Airtable Interface eingebettet werden.
